Sobre o Projeto

Este projeto foi desenvolvido para a ADO 2 da disciplina de Projeto Integrador II.

O objetivo é criar um CRUD em Angular para gerenciar livros de uma biblioteca, utilizando o JSON Server para simular uma API REST.

Funcionalidades

  • Listagem de livros cadastrados
  • Cadastro de novos livros
  • Edição de livros existentes
  • Exclusão de livros
  • Integração com API fake usando JSON Server
